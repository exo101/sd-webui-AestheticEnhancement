## 📖 插件介绍

美学提升插件是一个为 AI 绘画创作者设计的视觉参考工具，提供经典的构图法则和打光技巧素材库，帮助你快速提升作品的美学品质。

### ✨ 核心功能

- **📐 构图技巧**: 12 种经典构图示例（S 型、三角形、九宫格等）
- **💡 打光技巧**: 11 种专业光位演示（伦勃朗光、丁达尔光、逆光等）
- **🎬 AI 智能分析** (可选): Qwen3.5-VL 多模态大模型分析
  - 需要安装 Ollama 和 Qwen3.5 模型
  - 支持图片和视频分析
  - 自动识别构图、灯光、分镜等元素
- **🔍 点击放大**: 支持全屏查看高清细节
- **📱 响应式设计**: 完美适配各种屏幕尺寸

### 🎯 适用场景

- AI 绘画提示词参考
- 摄影构图学习
- 灯光布置灵感
- 美术基础训练

### 🎬 AI 智能分析（可选功能）

**注意**: 此功能需要额外配置 Ollama 和 Qwen3.5 模型

#### 前置要求

1. **安装 Ollama**: 从 https://ollama.com 下载并安装
2. **安装 Qwen3.5 模型**: 运行命令 `ollama run qwen3.5:4b`
3. **安装 Python 依赖**: `pip install requests opencv-python`
4. **启动 Ollama 服务**: 确保 Ollama 在后台运行

#### 使用步骤

1. 切换到 **🎬 AI 智能分析** 标签页
2. 点击"🔌 测试 Ollama 连接"确认服务正常
3. 选择分析模式（图片/视频）
4. 选择分析类型：
   - 🔍 综合分析
   - 📐 仅构图分析
   - 💡 仅灯光分析
   - 🎥 仅分镜分析
5. 上传图片或视频
6. 点击"🚀 开始分析"
7. <img width="1789" height="916" alt="QQ20260328-233545" src="https://github.com/user-attachments/assets/11c2bd6a-a25f-48d3-88ba-d0c842fa5136" />


#### 功能特点

- **图片分析**: 自动识别构图、灯光、色彩、镜头语言
- **视频分析**: 智能镜头检测，关键帧提取，批量分析
- **多模型支持**: qwen3.5:2b/4b/9b，平衡速度与质量
- **完全免费**: 本地运行，无需 API Key

### 使用技巧

1. **提示词参考**: 根据构图或光位名称生成提示词，如 "rule of thirds composition", "rembrandt lighting"
2. **对比学习**: 对比不同构图的视觉效果，理解其情感表达差异
3. **实践应用**: 在创作时有意识地运用学到的构图和布光技巧

## ❓ 常见问题

### Q: 图片无法加载？

A: 请检查以下事项：
1. 确认 `Aesthetic-Enhancement` 目录存在于插件文件夹中
2. 确认目录下有 `构图技巧` 和 `打光技巧` 两个子目录
3. 检查图片文件是否完整（PNG 格式）

### Q: 点击卡片没有反应？

A: 可能是 JavaScript 未正确加载，尝试：
1. 刷新浏览器页面（Ctrl+F5 强制刷新）
2. 清除浏览器缓存
3. 检查浏览器控制台是否有报错（F12）

### Q: 如何添加到提示词？

A: 参考以下方式：
- 构图：`rule of thirds composition`, `symmetrical composition`, `leading lines`
- 灯光：`rembrandt lighting`, `backlight`, `side light`, `volumetric lighting`

### Q: 可以自定义素材吗？

A: 可以！直接将你的参考图片放入对应的素材目录即可：
- 构图图片放入 `Aesthetic-Enhancement/构图技巧/`
- 打光图片放入 `Aesthetic-Enhancement/打光技巧/`
